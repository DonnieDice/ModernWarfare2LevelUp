## <span style="color: #4F4F4F;">✨ Features</span>

<span style="color: #4F4F4F;">🎵 <strong>MW2 Sound</strong></span> <span style="color: #3598db;">-</span> <span style="color: #e67e23;">A Modern Warfare 2-inspired level-up sound plays every time you gain a level.</span>

<span style="color: #4F4F4F;">🎚️ <strong>Quality Options</strong></span> <span style="color: #3598db;">-</span> <span style="color: #e67e23;">Switch between</span> <span style="color: #4ecdc4;">High</span><span style="color: #e67e23;">,</span> <span style="color: #4ecdc4;">Medium</span><span style="color: #e67e23;">, and</span> <span style="color: #4ecdc4;">Low</span> <span style="color: #e67e23;">quality variants.</span>

<span style="color: #ff6b6b;">🔇 <strong>Smart Muting</strong></span> <span style="color: #3598db;">-</span> <span style="color: #e67e23;">Auto-mutes WoW's default level-up sound so you only hear MW2LU.</span>

<span style="color: #4F4F4F;">💬 <strong>Login Message</strong></span> <span style="color: #3598db;">-</span> <span style="color: #e67e23;">A branded MW2LU login message appears on login and can be toggled on or off.</span>

<span style="color: #b96ad9;">💾 <strong>Persistent Settings</strong></span> <span style="color: #3598db;">-</span> <span style="color: #e67e23;">Settings are saved between sessions.</span>

<span style="color: #4ecdc4;">💬 <strong>Slash Commands</strong></span> <span style="color: #3598db;">-</span> <span style="color: #e67e23;">Simple</span> <span style="color: #4ecdc4;">/mw2lu</span> <span style="color: #e67e23;">commands for testing and switching sound quality.</span>

<span style="color: #4F4F4F;">🎮 <strong>Multi-Version</strong></span> <span style="color: #3598db;">-</span> <span style="color: #e67e23;">Includes support for Retail, Wrath, Burning Crusade, and Classic Era.</span>

## <span style="color: #4F4F4F;">📋 Slash Commands</span>

<span style="color: #e67e23;">The addon can be managed using the following slash commands</span><span style="color: #3598db;">:</span>

<div style="background-color: #0d1117; color: #58a6ff; font-family: 'Consolas', 'Monaco', monospace; border: 1px solid #30363d;">/mw2lu help     # Show all available commands<br>/mw2lu test     # Play test sound<br>/mw2lu enable   # Enable the addon<br>/mw2lu disable  # Disable the addon<br>/mw2lu welcome  # Toggle login message<br>/mw2lu high     # Use high quality sound<br>/mw2lu med      # Use medium quality sound (default)<br>/mw2lu low      # Use low quality sound</div>
