1. **<span style="color:#2dc26b">Install</span>** <span style="color:#d0f561">M</span><span style="color:#fff">odern Warfare 2 </span><span style="color:#d0f561">L</span><span style="color:#fff">evel-</span><span style="color:#d0f561">U</span><span style="color:#fff">p</span><span style="color:#d0f561">!</span> <span style="color:#e67e23">from your preferred platform</span>

3. **<span style="color:#b96ad9">Restart</span>** <span style="color:#06c">World of Warcraft</span>

<a id="command-reference"></a>
## <span style="color:#d0f561">📋 Command Reference</span>

<span style="color:#e67e23">Use</span> <span style="color:#2dc26b">`/mw2lu`</span> <span style="color:#e67e23">followed by</span><span style="color:#3598db">:</span>

<div align="center">

| Command | Description |
|---------|-------------|
| `/mw2lu help` | Show all available commands |
| `/mw2lu test` | Play test sound |
| `/mw2lu enable` | Enable addon |
| `/mw2lu disable` | Disable addon (unmutes default sound) |
| `/mw2lu welcome` | Toggle login welcome message |
| `/mw2lu high` | Use high quality sound |
| `/mw2lu med` | Use medium quality sound (default) |
| `/mw2lu low` | Use low quality sound |

</div>
