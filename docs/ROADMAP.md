# ModernWarfare2LevelUp Roadmap

ModernWarfare2LevelUp is currently focused on release hygiene, naming consistency, and keeping parity with the other RGX sound addons.

## Current Focus

1. Release pipeline stability
- Keep package metadata, TOC files, README, and CurseForge description aligned.
- Maintain tag-driven packaging through GitHub Actions.

2. Addon consistency
- Keep MW2LU structure aligned with SRLU and FFLU where it makes sense.
- Maintain clean addon naming across repo, package, sound files, and runtime paths.

3. Gameplay polish
- Validate Modern Warfare 2-themed sound behavior across supported WoW client versions.
- Keep slash commands and saved settings lightweight and predictable.
